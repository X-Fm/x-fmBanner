## 📦 Installation (Termux)

```bash
apt update && apt upgrade -y
```

```bash
pkg install git -y
```

```bash
git clone https://github.com/X-Fm/x-fmBanner.git
```

```bash
cd x-fmBanner/
```

```bash
chmod +x *
```

```bash
bash install.sh
```

### ▶️ Run

```bash
exit
```

### ⚡ One-Line Install

```bash
apt update && apt upgrade -y ; pkg install git -y ; git clone https://github.com/X-Fm/x-fmBanner.git ; cd x-fmBanner/ ; chmod +x * ; bash install.sh
```
